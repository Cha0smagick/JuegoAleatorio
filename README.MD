# Juego aleatorio 1.0
# by: Cha0smagick
# Sigueme en twitch: https://www.twitch.tv/cha0smagick

Este es un ejercicio que hace algo simple: la maquina genera un numero aleatorio. Luego, el jugador debe adivinar el numero, de  tal manera que 
el programa te va diciendo si el numero es mayor o menor. Cuando lo logras, el programa arroja el nunmero de intentos que utilizo
el usuario para poder descubrirlo. 

Si se descomenta la linea de impresion, imprime el numero random, esto con el fin de realizar pruebas.
